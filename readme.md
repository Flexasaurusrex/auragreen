3. **Deploy to Vercel:**

   - Click "New Project"
   - Import your GitHub repository
   - Vercel will auto-detect it's a static site
   - Click "Deploy"
   - Your site will be live in ~30 seconds!

1. **Upload images to your repository:**

   ```html

   </div>
   ```

- `#1a1a1a` - Dark gray cards
